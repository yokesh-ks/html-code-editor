# Contributing to HTML Code Editor

Thank you for your interest in contributing to the HTML Code Editor project! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository** - Click the fork button on GitHub to create your own copy
2. **Clone your fork** - Use `git clone` to download your forked repository
3. **Create a branch** - Create a new branch for your feature: `git checkout -b feature/your-feature-name`
4. **Make changes** - Implement your changes or bug fixes
5. **Commit changes** - Use clear, descriptive commit messages
6. **Push to your fork** - Push your branch to your forked repository
7. **Submit a pull request** - Open a PR with a description of your changes

## Code Guidelines

- Follow the existing code style and conventions
- Write clean, readable, and well-commented code
- Add tests for new features when applicable
- Update documentation if needed
- Ensure your code passes linting checks

## Reporting Issues

If you find a bug or have a suggestion:
- Check if the issue already exists
- Provide a clear description
- Include steps to reproduce (for bugs)
- Add relevant screenshots or examples

## Code of Conduct

Please note that this project is governed by our Code of Conduct. By participating, you are expected to uphold this code.

## Questions?

Feel free to open an issue or contact us if you have any questions about contributing.

Happy coding!
